Renoise Tools by Cas

This project consists of a handful (40+) of tools to make operating music making software Renoise easier. Short descriptions for all of these are also available, on my web page.

How to install

You can install, and update, all tools right into your Renoise user-scripts folder in a *nix-based OS:

$ cd ~/.renoise/V2.8.1/Scripts/Tools
$ git clone github.com/casmarrav/renoise-tools

Then git will make a folder 'renoise-tools', you will have to manually move everything in there one level up, making sure you don't forget all the .dotfiles (.git dir, .gitignore). On Windows the same method can be used; the directory you need to look for is 

%HOMEDRIVE%%HOMEPATH%\AppData\Roaming\Renoise\V2.8.1\Scripts\Tools\

Of course, if you don't need the auto-updates, you can also download the zip and extract it into the same folder.

How to use

Some of the tools are hard to find inside Renoise once installed: these only add features accessible through keyboard commands. An example of this is the Instrument Mixer, a very handy command window that allows the Renoise user to view and adjust various settings of multiple samples in an instrument at once. A very 'invisible' tool is Auto Select Instrument. It is no more than a variation on the function that is available in Renoise - in that it doesn't switch instruments on a pattern advance/change, only on track change. You can set a custom keyboard shortcut for toggling its functionality on and off in Renoises preferences window.
